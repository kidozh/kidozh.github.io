暑假安排

发表于 2017-06-18

前言

鉴于暑假时间特别的长，再加上我的个人原因，我写了一个大致的计划来规划我暑假的生活。

暑假的安排主要还是面向以后的方向，在暑假主要是查阅文献并整理出一个文献的报告。

时间长度

按照学校的安排，我主要还是按照一周来计划要做的事情

周内安排

在最终期限的早上，我会把我的一些对于本周的阅读的文献还有心得体会写到我的个人博客里，同时也力求提出一些point来启发我进行接下来的阅读
